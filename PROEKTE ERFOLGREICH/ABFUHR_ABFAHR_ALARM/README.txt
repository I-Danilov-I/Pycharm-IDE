Das Programm lädt den aktuellen Abfuhrkalender von dem als Link eingegebenen Bezirk.
Liest und sortiert die Termine, versendet eine E-Mail, wenn eine Tonne abgeholt wird.
Solange sich das Format von der Abfallkalender Datei nicht ändern,
werden die Termine unabhängig davon, ob sie von der Stadt gemeine geändert wurde ausgelesen.
Das gleiche Funktioniert mit anderen Gemeinden, solange das PDF Format ähnlich ist.
